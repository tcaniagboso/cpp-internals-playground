# C++ Internals Playground

A learning-focused C++20 repository where I rebuild small versions of core containers, iterators, and object-model features from scratch.

The goal is not to replace the STL. The goal is to better understand how modern C++ works beneath the surface, including object lifetime, ownership, copy/move semantics, templates, iterators, and memory management.

---

## Topics

- Constructors and destructors
- Copy and move semantics
- Deleted functions
- RAII and ownership
- Dynamic arrays
- Linked lists
- Custom iterators
- Const and reverse iteration
- Range-based for loop support
- Basic memory management
